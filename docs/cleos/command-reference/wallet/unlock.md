## Description
Unlocks a wallet.
## Command
To unlock a wallet, specify the password provided when it was created.

```shell
$ ./cleos wallet unlock -n second-wallet --password PW5Ji6JUrLjhKAVn68nmacLxwhvtqUAV18J7iycZppsPKeoGGgBEw
```

## Output


```shell
Unlocked: 'second-wallet'
```
