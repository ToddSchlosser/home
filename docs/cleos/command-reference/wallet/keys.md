## Description
Lists the private keys from all unlocked wallets in wif format.
## Command


```shell
$ ./cleos wallet keys
```

```shell
[[
    "EOS6MRyAjQq8ud7hVNYcfnVPJqcVpscN5So8BhtHuGYqET5GDW5CV",
    "5KQwrPbwdL6PhXujxW37FSSQZ1JiwsST4cqQzDeyXtP79zkvFD3"
  ]
]
```
